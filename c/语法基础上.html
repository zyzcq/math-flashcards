第 1 节 · 注释基本使用

一句话理解

注释就是写给人看的解释文字，程序不会执行它。

注释的两种写法

// 单行注释，用于短说明

/*
    多行注释
    用于较长的说明
*/

使用建议：

  • 说明一句话能写完 → 用 //
  • 说明需要好几行 → 用 /* */

在 Hello World 中看注释的作用

#include <stdio.h>          // 预处理：引入标准输入输出头文件

int main()                  // main 函数：程序入口
{
    printf("Hello World");  // 输出一行文字
    return 0;               // 程序正常结束
}

这里的 return 0 中：

  • return 表示返回
  • 0 表示正常结束
  • 0 要和前面的 int 配对（后面学）

VS 编辑小技巧

操作 方法
临时调字体大小 Ctrl + 鼠标滚轮
永久改字体 工具 → 选项 → 环境 → 字体和颜色
推荐代码字体 Consolas
批量编辑多行 按住 Alt + 鼠标左键竖选
本节精炼：注释 = 给人看的说明；C 语言用 // 和 /* */ 两种注释，对程序运行没有影响。

第 2 节 · 注释扩展知识点

1. 注释快捷键（最常用）

功能 快捷键
添加注释 Ctrl + K，再按 Ctrl + C
取消注释 Ctrl + K，再按 Ctrl + U
注意：不是同时按，是先按 Ctrl + K，再按第二个组合键。

操作方式：

  • 单行：光标点到该行 → 快捷键
  • 多行：先选中多行 → 快捷键

快捷键只会帮你加 //，不会帮你写注释内容。

2. 注释擦除

.c 文件（代码 + 注释）
     ↓ 编译
.obj 文件（只剩代码，注释被擦除）

结论：

  • 注释不参与编译
  • 注释不参与运行
  • 注释里写什么都不会影响结果
但是：注释一定要和代码有关，不要写吐槽或无关内容。

3. 注释嵌套规则

// 单行注释里再写 //  ✅ 没问题

/*
    多行注释里写 //  ✅ 没问题
*/

/*
    外层多行注释
    /*
        内层多行注释  ❌ 会报错
    */
*/

原因：

多行注释从 /* 开始，会匹配最近的 */ 作为结束，嵌套时外层会提前结束，导致后面多出的 */ 语法错误。
本节精炼：记住 Ctrl + K, Ctrl + C 加注释；注释不影响运行；多行注释不要嵌套多行注释。

第 3 节 · 关键字

一句话理解

关键字 = C 语言规定好、有特殊含义的英文单词。

已经见过的关键字

int       // 表示整数相关含义
return    // 表示返回，常用于结束函数

C 语言总共有 32 个关键字，现在不用背，用到哪个学哪个。

如何识别关键字

两个特征：

  1. 全部小写（C 语言区分大小写）
  2. 在编辑器中有特殊颜色
    • VS：蓝色 / 紫色
    • VC：蓝色
int      ✅ 是关键字
Int      ❌ 不是
INT      ❌ 不是

关键字不能当变量名

// ❌ 错误写法
int int = 10;
int return = 20;

// ✅ 正确写法
int age = 18;
int score = 100;
本节精炼：关键字是 C 语言保留词，全小写 + 高亮显示；不能拿来当变量名。

第 4 节 · 常量

一句话理解

常量 = 程序运行中值不会改变的数据。

四种常量一览

类型 含义 写法 例子
整型常量 整数 直接写 10、-20、0
实型常量 小数 带小数点 3.14、18.2
字符常量 单个字符 用单引号 'a'、'1'、'+'、' '
字符串常量 一串文本 用双引号 "abc"、"你好"、""

字符常量的规则（最容易错）

单引号里必须有 且只能有一个 合法字符，可以是：

  • 英文字母：'a'、'B'
  • 数字字符：'1'
  • 英文符号：'+'
  • 空格字符：' '

❌ 常见错误

''       // 错：里面什么都没有
'ab'     // 错：超过一个字符
'中'     // 错：不能写中文
'￥'     // 错：中文输入法下的符号

字符串常量的规则（非常宽松）

"abc"        // 多个字符
"你好"       // 中文
"123"        // 数字当文本
""           // 空字符串（合法）

三组易混对比

1       // 整型常量，可以计算
'1'     // 字符常量，表示字符 1
"1"     // 字符串常量，表示文本 1
123     // 数字，能做加减乘除
"123"   // 文本，不能直接计算
''      // ❌ 错误
""      // ✅ 合法的空字符串

判断常量类型的套路

有没有引号？

├─ 没引号
│   ├─ 整数 → 整型常量
│   └─ 小数 → 实型常量
├─ 单引号
│   ├─ 一个合法字符 → 字符常量
│   └─ 否则 → 错误
└─ 双引号 → 字符串常量
本节精炼：常量有 4 类：整型、实型、字符、字符串；单引号放一个字符，双引号放一串文本。

第 5 节 · 实型常量细节补充

扩展点 1：省略写法

0.93   ↔   .93    // 小数点前的 0 可省略
18.0   ↔   18.    // 小数点后的 0 可省略
⚠️ 能这样写，但不建议。可读性差，平时写完整即可。

扩展点 2：科学计数法

C 语言中的科学计数法要用 e 或 E：

1.234e7     // 表示 1.234 × 10^7 = 12340000
3.14e2      // 表示 3.14 × 10^2
2e4         // 表示 2 × 10^4
1.234E7     // 大写 E 也可以

这类写法属于 实型常量。

⚠️ 特别容易踩的坑

12340000        // 整型常量
1.234e7         // 实型常量（科学计数法）
1.234 * 10^7    // 既不是整型也不是实型，而且 ^ 在 C 语言里不是乘方！

科学计数法中间 不能有空格：

1.23e4     ✅
1.23 e4    ❌

常量真题判断口诀

判断什么 关键规则
整型常量 必须是整数，无小数点，中间无符号/空格
实型常量 有小数点，或使用 e/E 的科学计数法
科学计数法 中间不能有空格
能否数学计算 要用数字，不用 "字符串"
本节精炼：省略 0 能写但别这么写；科学计数法用 e；1.234 * 10^7 不是常量是表达式。

第 6 节 · 输出常量

printf 的结构

printf("输出样式", 填补内容);
  • 第一个参数：必须是字符串，表示最终输出的样式
  • 第二个参数：可选，用来填补占位符

占位符（最核心概念）

先占位置，再用真实数据填进去。
类型 占位符 来源（了解即可）
整数 %d decimal
小数 %f floating point
字符 %c character
字符串 %s string

四种常量的输出写法

#include <stdio.h>

int main()
{
    printf("%d", 18);          // 输出整数：18
    printf("%f", 1.93);        // 输出小数：1.930000（默认 6 位）
    printf("%c", 'a');         // 输出字符：a
    printf("%s", "Hello");     // 输出字符串：Hello
    return 0;
}

结合文字一起输出

printf("我的年龄是%d岁", 18);
// 输出：我的年龄是18岁

printf("我的身高是%f米", 1.93);
// 输出：我的身高是1.930000米

为什么必须用占位符？——看这组对比

printf("18 + 18");        // 输出文本：18 + 18
printf("%d", 18 + 18);    // 输出计算结果：36

字符串里的数字只能展示，不能计算；占位符能帮你输出真实的计算结果。

字符串输出的两种方式

// 方式一：直接输出
printf("Hello World");

// 方式二：使用占位符
printf("我的名字是%s", "阿伟");

⚠️ VS 实操常见坑：多个 main

一个项目里只能有一个 main 函数，否则会报：

main 已经在 xxx.obj 中定义过

解决办法：

  1. 新建一个新项目；或
  2. 把旧的 main 注释掉、改名

两个练习

练习 1：输出学校名字（字符串 → %s）

#include <stdio.h>

int main()
{
    printf("我学校的名字是：%s", "罗家山职业技术学院");
    return 0;
}

练习 2：输出高考总分（小数 → %f）

#include <stdio.h>

int main()
{
    printf("我高考的总分是：%f", 666.0);
    // 输出：我高考的总分是：666.000000
    return 0;
}
本节精炼：printf 的第一个参数是字符串样式；用 %d / %f / %c / %s 当占位符，再用真实数据填进去。

六节课总复盘

三大知识块一张图

C 语言核心语法

├── 注释
│   ├── //         单行
│   ├── /* */      多行
│   ├── 不参与编译和运行
│   └── 快捷键：Ctrl+K, Ctrl+C / U
│
├── 关键字
│   ├── C 语言保留的特殊单词（32 个）
│   ├── 全小写 + 编辑器高亮
│   └── 不能当变量名
│
└── 常量（值不变的数据）
    ├── 整型：10、-5、0
    ├── 实型：3.14、18.2、.93、18.、1.234e7
    ├── 字符：'a'、'1'、'+'、' '（只能一个字符）
    ├── 字符串："abc"、"你好"、""
    └── 输出：printf + %d / %f / %c / %s

最该记住的 10 句话

  1. 注释是给人看的，不参与运行。
  2. 单行 //，多行 /* */，多行注释不要嵌套。
  3. Ctrl + K, Ctrl + C 加注释，Ctrl + K, Ctrl + U 取消。
  4. 关键字是 C 语言保留词，全小写且高亮，不能当变量名。
  5. 常量分 4 类：整型、实型、字符、字符串。
  6. 单引号只放一个合法字符，双引号可以放一串任意内容。
  7. 1、'1'、"1" 类型完全不同。
  8. 科学计数法用 e/E，中间不能有空格；^ 不是乘方。
  9. printf 第一个参数必须是字符串样式。
  10. 占位符：%d 整数、%f 小数、%c 字符、%s 字符串。

🎯 最终记忆句

注释让代码更好读，关键字是 C 的保留词，常量是不变的数据，printf 靠占位符把它们打印出来。
