1. 无限循环

概念无限循环就是循环条件一直成立，循环会持续执行，不会自动停止。

写法 格式 特点
for 无限循环 for (;;) 三个部分都省略，但两个分号不能省略。
while 无限循环 while (1) 最常用。C 语言中 0 为假，非 0 为真。
do while 无限循环 do { } while (1); 也能实现，但实际开发中较少用。
重点：无限循环通常不会单独使用，而是配合 if + break，在满足条件时退出。 
#include <stdio.h>

int main()
{
    while (1)
    {
        printf("正在循环\n");

        if (/* 结束条件 */)
        {
            break;
        }
    }

    printf("循环结束\n");
    return 0;
}
注意：如果无限循环中没有退出逻辑，那么循环后面的代码永远执行不到。手动停止程序可以关闭窗口或按 Ctrl + C。
本节精炼总结

无限循环有三种写法，最常用的是 while(1)。它的价值不是让程序卡死，而是让程序“先一直执行，满足结束条件时再退出”。

2. break

跳出break 表示结束或跳出。它只能写在 switch 或循环中。

在循环中遇到 break，会立刻结束当前循环，然后继续执行循环后面的代码。 
#include <stdio.h>

int main()
{
    for (int i = 1; i <= 100; i++)
    {
        if (i == 15)
        {
            break;
        }

        printf("%d\n", i);
    }

    printf("循环结束\n");
    return 0;
}

例题：找 1 到 100 中第一个既能被 3 又能被 5 整除的数

#include <stdio.h>

int main()
{
    for (int i = 1; i <= 100; i++)
    {
        if (i % 3 == 0 && i % 5 == 0)
        {
            printf("%d\n", i);
            break;
        }
    }

    return 0;
}
有 break：只输出第一个满足条件的数 15。
没有 break：会输出 15、30、45、60、75、90。
本节精炼总结

题目出现“第一个”“找到就停止”“后面不用继续”，优先想到 break。

3. continue

跳过本次continue 表示结束本次循环，继续下一次循环。

关键字 作用 使用范围
break 结束整个当前循环 switch 或循环
continue 跳过本次循环，继续下一次 只能用于循环

例题：吃 5 个包子，跳过第 3 个

#include <stdio.h>

int main()
{
    for (int i = 1; i <= 5; i++)
    {
        if (i == 3)
        {
            continue;
        }

        printf("正在吃第%d个包子\n", i);
    }

    return 0;
}
while 中使用 continue 要小心：如果 continue 跳过了变量更新，可能死循环。 
// 错误示例：i == 3 时会一直 continue，i 无法增加
int i = 1;
while (i <= 5)
{
    if (i == 3)
    {
        continue;
    }

    printf("%d\n", i);
    i++;
}
本节精炼总结

break 是“彻底不循环了”，continue 是“这一次跳过，下一次继续”。

4. 循环嵌套与打印矩形

结构循环嵌套就是循环里面还有循环。外面的叫外循环，里面的叫内循环。

打印图形时，一般：外循环控制行数，内循环控制每一行打印多少个。

打印三行五列矩形

#include <stdio.h>

int main()
{
    for (int i = 1; i <= 3; i++)
    {
        for (int j = 1; j <= 5; j++)
        {
            printf("*");
        }

        printf("\n");
    }

    return 0;
}

打印七行八列矩形

#include <stdio.h>

int main()
{
    for (int i = 1; i <= 7; i++)
    {
        for (int j = 1; j <= 8; j++)
        {
            printf("*");
        }

        printf("\n");
    }

    return 0;
}
阅读嵌套循环的技巧：先看内循环做什么，再看外循环让这件事重复几次。
本节精炼总结

矩形公式：外循环写行数，内循环写列数，内循环结束后换行。

5. 打印直角三角形

三角形与矩形的区别在于：每一行星号数量不同。所以通常改内循环。

倒着的五行直角三角形

每行星号数量：5、4、3、2、1。

#include <stdio.h>

int main()
{
    for (int i = 1; i <= 5; i++)
    {
        for (int j = i; j <= 5; j++)
        {
            printf("*");
        }

        printf("\n");
    }

    return 0;
}

正着的五行直角三角形

每行星号数量：1、2、3、4、5。

#include <stdio.h>

int main()
{
    for (int i = 1; i <= 5; i++)
    {
        for (int j = 1; j <= i; j++)
        {
            printf("*");
        }

        printf("\n");
    }

    return 0;
}
图形 行数 内循环规律
矩形 固定 j <= 固定列数
倒三角 固定 j = i; j <= 5
正三角 固定 j = 1; j <= i
本节精炼总结

图形题先看“一共几行”，再看“第几行有几个”。外循环管行，内循环管每行数量。

6. 九九乘法表

应用九九乘法表本质是“正直角三角形 + 每个位置打印乘法表达式”。

写法三步：先打印九行九列矩形，再改成正三角形，最后把星号改成乘法表达式。 
#include <stdio.h>

int main()
{
    for (int i = 1; i <= 9; i++)
    {
        for (int j = 1; j <= i; j++)
        {
            printf("%d*%d=%d\t", j, i, j * i);
        }

        printf("\n");
    }

    return 0;
}
i 控制当前第几行，j 控制当前行中的第几个表达式。常见格式是 j * i = j * i。
如果写成 i, j, i * j，乘积没错，但显示顺序可能不是常见九九乘法表。运行后可根据结果调整 i 和 j 的位置。
本节精炼总结

九九乘法表核心：外循环 1 到 9，内循环 j <= i，每次打印 %d*%d=%d\t。

7. 制表符 \t

格式对齐\t 是制表符，可以理解成长度可变的大空格。

它会根据前面内容长度补空格，让整体长度达到 8 或 8 的倍数。最少补 1 个，最多补 8 个。
内容 前面长度 \t 补空格结果
ABC\t 3 补 5 个，到 8
长度刚好 8 后接 \t 8 补 8 个，到 16
一个中文后接 \t 通常可理解为 2 补到 8，具体显示受环境影响

表格对齐示例

#include <stdio.h>

int main()
{
    printf("name\t\tage\n");
    printf("张三\t23\n");

    return 0;
}
\t 不是永远自动完美对齐。如果不同内容长度差距较大，可能需要增加或减少制表符数量。
本节精炼总结

\n 是换行，\t 是横向对齐。打印九九乘法表、姓名年龄表等表格数据时常用。

8. 统计质数

算法质数是只能被 1 和自身整除的整数。1 不是质数，所以统计时通常从 2 开始。

复杂题拆解：先判断一个数是不是质数，再扩大到范围，最后统计数量。

判断一个数是否为质数

#include <stdio.h>

int main()
{
    int number = 17;
    int count = 0;

    for (int i = 2; i < number; i++)
    {
        if (number % i == 0)
        {
            count++;
            break;
        }
    }

    if (count == 0)
    {
        printf("%d是质数\n", number);
    }
    else
    {
        printf("%d不是质数\n", number);
    }

    return 0;
}

统计小于等于 n 的质数数量

#include <stdio.h>

int main()
{
    int n = 100;
    int countA = 0;

    for (int i = 2; i <= n; i++)
    {
        int count = 0;

        for (int j = 2; j < i; j++)
        {
            if (i % j == 0)
            {
                count++;
                break;
            }
        }

        if (count == 0)
        {
            countA++;
        }
    }

    printf("小于等于%d的质数数量是：%d\n", n, countA);

    return 0;
}
count 判断当前数字是否有因数；countA 统计一共有多少个质数。
内循环条件写 j < i，不要写 j <= i，因为任何数都能被自己整除。
本节精炼总结

外循环拿数字，内循环判断是否质数，外部计数器统计总数。找到一个因数后立刻 break，提高效率。

9. 幂级数列

累乘 + 累加题目：计算 1 的 1 次方加 2 的 2 次方，一直加到 10 的 10 次方。

也就是计算：

1^1 + 2^2 + 3^3 + ... + 10^10

结果较大，变量不要用 int，要用 long long。打印时用 %lld。 
#include <stdio.h>

int main()
{
    long long res = 0;

    for (int i = 1; i <= 10; i++)
    {
        long long power = 1;

        for (int j = 1; j <= i; j++)
        {
            power *= i;
        }

        res += power;
    }

    printf("%lld\n", res);

    return 0;
}
变量 作用 初始值原因
power 保存当前项，比如 3^3 乘法累积从 1 开始
res 保存最终总和 加法累积从 0 开始
res += power 必须写在内循环结束后。否则会把中间结果也累加进去。
本节精炼总结

外循环决定算哪一项，内循环计算这一项的幂，内循环结束后再把这一项加到总结果中。

10. 特殊数字：各位数字之和等于 15

拆数字题目：找出范围内每一位数字之和等于 15 的所有数字。

78：7 + 8 = 15，满足。
168：1 + 6 + 8 = 15，满足。
1167：1 + 1 + 6 + 7 = 15，满足。
通用套路：外循环获取范围中的每个数字；内循环拆出这个数字的每一位并累加；最后判断是否等于 15。

拆数字核心

int temp = number % 10;  // 获取最后一位
number = number / 10;   // 去掉最后一位

完整代码

#include <stdio.h>

int main()
{
    int n = 1000;

    for (int i = 0; i <= n; i++)
    {
        int number = i;
        int sum = 0;

        while (number != 0)
        {
            int temp = number % 10;
            sum += temp;

            number /= 10;
        }

        if (sum == 15)
        {
            printf("%d\n", i);
        }
    }

    return 0;
}
不要直接修改外循环变量 i。要先复制一份：int number = i;，再用 number 去拆位。
文稿中范围出现过 0 到 100、0 到 1000，例子又出现 1167。实际写法是通用的，想包含更大数字，只需要修改 n。
本节精炼总结

拆数字靠 % 10 和 / 10。外循环取数，内循环拆位求和，最后判断 sum == 15。

11. 跳出多层循环与 goto

控制流程嵌套循环中，如果 break 写在内循环里，它默认只跳出内循环。

break 只跳出当前层循环

#include <stdio.h>

int main()
{
    for (int i = 1; i <= 3; i++)
    {
        for (int j = 1; j <= 5; j++)
        {
            printf("内循环执行%d\n", j);
            break;
        }

        printf("内循环结束\n");
        printf("----------\n");
    }

    return 0;
}
结果会出现 3 组，每组只打印一次“内循环执行1”。说明 break 只跳出了内循环，外循环仍然继续。

goto + 标号：跳出多层循环

#include <stdio.h>

int main()
{
    for (int i = 1; i <= 3; i++)
    {
        for (int j = 1; j <= 5; j++)
        {
            printf("内循环执行%d\n", j);
            goto end;
        }

        printf("内循环结束\n");
        printf("----------\n");
    }

end:
    printf("所有循环都结束了\n");

    return 0;
}

标号格式：

标号名:

跳转格式：

goto 标号名;
goto 可以跳到代码中任意位置，非常灵活，但会降低代码可读性。一般不要乱用。

不推荐：用 goto 写无限循环

#include <stdio.h>

int main()
{
    int i = 1;

a:
    printf("你好%d\n", i);
    i++;

    goto a;

    return 0;
}

更推荐：while(1) + if + break

#include <stdio.h>

int main()
{
    int i = 1;

    while (1)
    {
        printf("你好%d\n", i);
        i++;

        if (i == 15)
        {
            break;
        }
    }

    printf("看看我执行了吗？\n");

    return 0;
}
本节精炼总结

break 只能跳出当前循环。跳出多层循环可以使用 goto + 标号，但普通循环退出优先用 break。
