一、自增自减运算符基础

1. 什么是自增、自减

++ 叫做自增运算符，作用是让变量的值加 1。

int a = 10;
a++;    // a 变成 11

-- 叫做自减运算符，作用是让变量的值减 1。

int a = 10;
a--;    // a 变成 9
重点：a++ 等价于 a = a + 1，a-- 等价于 a = a - 1。

2. 四种基本写法

写法 名称 含义
a++ 后加加 让 a 加 1
++a 先加加 让 a 加 1
a-- 后减减 让 a 减 1
--a 先减减 让 a 减 1
正确理解：如果它们单独写成一行，前置和后置效果一样。 
int a = 10;

a++;    // a = 11
++a;    // a = 12
a--;    // a = 11
--a;    // a = 10

所以上面代码执行完后，a 最终还是 10。

3. 实际应用场景

自增自减常用在需要“加 1”或“减 1”的地方。

场景一 年龄增长：

int age = 1;
age++;    // 过生日后变成 2

场景二 购物车数量变化：

int count = 1;
count++;  // 点击加号，数量加 1
count--;  // 点击减号，数量减 1

场景三 数据统计：

int total = 0;

total++;  // 每发现一个符合条件的数据，就加 1
本节重点：自增自减最常用的写法是单独成行，尤其适合计数、统计、数量变化。

二、自增自减参与计算

1. 什么是参与计算

如果自增自减不再单独写一行，而是放在表达式里，就叫参与计算。

int a = 10;
int b = a++;

或者：

int a = 10;
int b = ++a;
易错点：参与计算时，a++ 和 ++a 的结果不一样。

2. 后加加：先用后加

a++ 参与计算时，规则是：先使用原来的值，再让变量加 1。

int a = 10;
int b = a++;

执行过程：

步骤 结果
先把 a 原来的值 10 赋值给 b b = 10
然后 a 自增 a = 11
口诀：a++ 是先用后加。

3. 先加加：先加后用

++a 参与计算时，规则是：变量先加 1，再拿新值参与计算。

int a = 10;
int b = ++a;

执行过程：

步骤 结果
a 先自增 a = 11
再把 a 的新值赋值给 b b = 11
口诀：++a 是先加后用。

4. 连续执行时要追踪变量最新值

int a = 10;

int b = a++;
int c = ++a;

分析：

代码 执行后结果
int a = 10; a = 10
int b = a++; b = 10，a = 11
int c = ++a; a = 12，c = 12
正确做法：每执行一行，都要更新变量当前值。下一行一定基于最新值继续计算。

5. 多个自增混在一个表达式中

文稿中讲了比较复杂的表达式，例如：

int result = i++ + ++i - --j - i--;
重要提醒：这类写法实际开发中几乎不用，因为可读性差，而且不同系统或编译环境可能结果不同。它主要用于考试选择题。

文稿中给出的规则：

环境 规则
Windows 前缀优先于后缀；前缀统一先计算；后缀统一先使用
Mac / Linux 前缀和后缀优先级一样，从左到右依次独立计算

Windows 规则例子

int i = 10;
int j = 5;

int result = i++ + ++i - --j - i--;

按照文稿中的 Windows 规则：

先处理前缀：++i 让 i 变成 11，--j 让 j 变成 4。

然后后缀统一先用：i++ 和 i-- 都先使用当前的 11。

result = 11 + 11 - 4 - 11;  // 结果是 7

Mac / Linux 规则例子

从左到右依次处理：

i++   使用 10，然后 i 变成 11
++i   i 变成 12，然后使用 12
--j   j 变成 4，然后使用 4
i--   使用 12，然后 i 变成 11

result = 10 + 12 - 4 - 12;  // 结果是 6
复习重点：考试中可能会考复杂表达式；实际写代码时不要这样写，应该拆开。

6. 文稿中的两组考试题结果

第一组，假设从 int a = 10; 开始，连续执行：

int k1 = a++ + a++;
int k2 = ++a + ++a;
int k3 = ++a + a++;
int k4 = a++ + ++a;

文稿中 Windows 规则结果：

表达式 结果
k1 = a++ + a++ 20
k2 = ++a + ++a 28
k3 = ++a + a++ 30
k4 = a++ + ++a 34

第二组，多了一个普通的 + a：

int k1 = a++ + a++ + a;
int k2 = ++a + ++a + a;
int k3 = ++a + a++ + a;
int k4 = a++ + ++a + a;

文稿中 Windows 规则结果：

表达式 结果
k1 = a++ + a++ + a 30
k2 = ++a + ++a + a 42
k3 = ++a + a++ + a 45
k4 = a++ + ++a + a 51
易错点：后缀自增不是前面用完就马上加，要等表达式中相关变量都用完后再自增。

三、赋值运算符

1. 普通赋值运算符：=

= 是赋值运算符，作用是把右边的值赋给左边的变量。

int a = 10;

意思是：把 10 赋值给变量 a。

易错点：= 是赋值，不是判断相等。判断相等要用 ==。

2. 扩展赋值运算符

除了 =，还有扩展赋值运算符：

运算符 含义 等价写法
+= 加后赋值 a = a + b
-= 减后赋值 a = a - b
*= 乘后赋值 a = a * b
/= 除后赋值 a = a / b
%= 取余后赋值 a = a % b
统一规则：先让左边变量和右边数据进行运算，再把结果赋值给左边变量。右边变量不受影响。

3. += 的例子

int a = 10;
int b = 20;

a += b;

a += b 等价于：

a = a + b;

所以：

a = 10 + 20;  // a = 30
b = 20;       // b 不变

4. *= 的例子

如果前面已经执行过：

int a = 10;
int b = 20;

a += b;  // a = 30

继续执行：

a *= b;

等价于：

a = a * b;

此时 a 已经是 30，所以：

a = 30 * 20;  // a = 600
b = 20;       // b 不变
正确理解：后面的计算要使用变量当前最新的值，不是最开始的值。

四、关系运算符

1. 什么是关系运算符

关系运算符也叫比较运算符，用来判断左右两边的关系是否成立。

a == b

意思是：判断 a 和 b 是否相等。

C语言判断结果：成立为 1，不成立为 0。

2. 六种关系运算符

运算符 含义 示例
== 判断是否相等 a == b
!= 判断是否不相等 a != b
> 判断是否大于 a > b
>= 判断是否大于等于 a >= b
< 判断是否小于 a < b
<= 判断是否小于等于 a <= b
易错点：判断相等是 ==，不是 =。大于等于是 >=，小于等于是 <=。

3. 基础练习

int a = 10;
int b = 20;

printf("%d\n", a == b);
printf("%d\n", a != b);
printf("%d\n", a > b);
printf("%d\n", a >= b);
printf("%d\n", a < b);
printf("%d\n", a <= b);

输出结果：

0
1
0
0
1
1

原因：

表达式 判断 结果
10 == 20 不成立 0
10 != 20 成立 1
10 > 20 不成立 0
10 >= 20 不成立 0
10 < 20 成立 1
10 <= 20 成立 1

4. 实际应用场景

判断偶数

int a = 10;
printf("%d\n", a % 2 == 0);

a % 2 == 0 表示判断 a 除以 2 的余数是否为 0。成立就是偶数，不成立就是奇数。

判断不超过 100

int number = 100;
printf("%d\n", number <= 100);

“不超过 100”就是小于等于 100，写作 <= 100。

判断银行卡余额是否足够

int pay = 100;
int money = 200;

printf("%d\n", pay <= money);

也可以写成：

printf("%d\n", money >= pay);

判断库存是否足够

int buy = 10;
int stock = 100;

printf("%d\n", buy <= stock);

也可以写成：

printf("%d\n", stock >= buy);

筛选不低于 100 元的商品

int goods = 500;

printf("%d\n", goods >= 100);

“不低于 100”就是大于等于 100，写作 >= 100。

本节重点：关系运算符经常和判断、筛选、条件控制结合使用。现在先记住它的结果是 1 或 0。

五、代码总整理

1. 自增自减基础代码

#include <stdio.h>

int main()
{
    int a = 10;

    a++;
    ++a;
    a--;
    --a;

    printf("a = %d\n", a);

    return 0;
}

结果：

a = 10

2. 自增自减参与计算代码

#include <stdio.h>

int main()
{
    int a = 10;

    int b = a++;
    int c = ++a;

    printf("a = %d\n", a);
    printf("b = %d\n", b);
    printf("c = %d\n", c);

    return 0;
}

结果：

a = 12
b = 10
c = 12

3. 赋值运算符代码

#include <stdio.h>

int main()
{
    int a = 10;
    int b = 20;

    a += b;

    printf("执行 a += b 后：\n");
    printf("a = %d\n", a);
    printf("b = %d\n", b);

    a *= b;

    printf("执行 a *= b 后：\n");
    printf("a = %d\n", a);
    printf("b = %d\n", b);

    return 0;
}

结果：

执行 a += b 后：
a = 30
b = 20
执行 a *= b 后：
a = 600
b = 20

4. 关系运算符代码

#include <stdio.h>

int main()
{
    int a = 10;
    int b = 20;

    printf("%d\n", a == b);
    printf("%d\n", a != b);
    printf("%d\n", a > b);
    printf("%d\n", a >= b);
    printf("%d\n", a < b);
    printf("%d\n", a <= b);

    int num = 10;
    printf("是否为偶数：%d\n", num % 2 == 0);

    int pay = 100;
    int money = 200;
    printf("余额是否足够：%d\n", pay <= money);

    int buy = 10;
    int stock = 100;
    printf("库存是否足够：%d\n", buy <= stock);

    int goods = 500;
    printf("商品是否不低于100元：%d\n", goods >= 100);

    return 0;
}

六、最终复习清单

必须掌握

1. 自增自减基础：

++ 表示加 1，-- 表示减 1。

单独成行时，a++ 和 ++a 效果一样；a-- 和 --a 效果一样。

2. 自增自减参与计算：

a++：先用后加。

++a：先加后用。

复杂表达式实际开发少用，主要用于考试。

3. 赋值运算符：

= 是赋值，把右边给左边。

+= -= *= /= %= 都是先运算，再赋值给左边变量。

4. 关系运算符：

== != > >= < <= 用来比较。

关系表达式成立结果是 1，不成立结果是 0。

最容易错的地方：

判断相等用 ==，不是 =。

大于等于是 >=，小于等于是 <=。

复杂自增表达式不要在实际开发中使用。
