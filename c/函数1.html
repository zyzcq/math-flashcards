C语言函数系统笔记

一、函数是什么

概念 函数就是把一段代码打包成一个独立功能，并给它起一个名字。以后想使用这段代码时，直接调用函数名即可。

函数的核心价值：减少重复代码，让程序结构更清晰，让代码更容易维护。

为什么需要函数

如果一段代码要反复使用，直接复制粘贴会导致代码重复、修改麻烦、容易出错。 函数可以把重复代码集中管理，需要时调用一次即可。

函数和循环的区别

对比 函数 循环
重点 功能打包 连续重复执行
适合场景 某个功能在不同位置复用 同一段代码连续执行多次
函数不是定义了就自动执行，必须调用才执行。

二、函数的基本格式

格式 最简单的函数格式如下： 

void 函数名()
{
    函数体;
}
组成 作用
void 暂时理解为没有返回值
函数名 给函数起的名字，方便以后调用
() 函数名后必须写，小括号不能省略
{} 包住函数体，表示哪些代码属于这个函数
函数体 函数真正要执行的代码

函数应该写在哪里

C语言中不能在一个函数里面再定义另一个函数。main 本身也是函数，所以自定义函数不能写在 main 里面。

推荐初学者先把自定义函数写在 main 上面：

#include <stdio.h>

void play_game()
{
    printf("选择人物\n");
    printf("准备开局\n");
    printf("游戏开始\n");
}

int main()
{
    play_game();

    return 0;
}

函数调用

调用函数的格式： 

函数名();
注意：调用函数时，函数名后面的小括号必须写。play_game(); 是调用函数，play_game; 不是调用函数。

三、无参数函数练习：求 10 和 20 的和

这一节的重点不是加法，而是练习：把求和逻辑写进函数，再在 main 中调用。 

#include <stdio.h>

void sum()
{
    int num1 = 10;
    int num2 = 20;
    int result = num1 + num2;

    printf("%d\n", result);
}

int main()
{
    sum();

    return 0;
}
运行结果是 30。函数定义只是准备功能，sum(); 才会真正执行函数。
当前写法的缺点：只能求 10 和 20 的和。如果想求别的数，需要修改函数内部代码，所以后面要学习带参数函数。

四、带参数的函数

为什么需要参数

如果函数中要处理的数据不固定，就应该把这些不确定的数据设计成参数。 例如求两个数的和，两个数每次可能不同，所以要通过参数传入。

带参数函数格式

void 函数名(参数类型 参数名1, 参数类型 参数名2)
{
    函数体;
}

例如：

void sum(int num1, int num2)
{
    int result = num1 + num2;
    printf("%d\n", result);
}

形参和实参

名称 位置 含义
形参 函数定义的小括号里 用来接收数据的变量
实参 函数调用的小括号里 真正传进去的数据 
void sum(int num1, int num2) // num1、num2 是形参
{
    int result = num1 + num2;
    printf("%d\n", result);
}

int main()
{
    sum(10, 20); // 10、20 是实参
    sum(30, 40); // 30、40 是实参

    return 0;
}
调用 sum(10, 20); 时，相当于把 10 赋值给 num1，把 20 赋值给 num2。

形参和实参必须一一对应

  • 形参有几个，实参就要传几个。
  • 少传不行，多传也不行。
  • 多个形参之间用英文逗号隔开。
  • 顺序要对应，尤其是减法、除法等对顺序敏感的运算。
错误示例：函数定义有两个形参，却只传一个实参。 
void sum(int num1, int num2)
{
    printf("%d\n", num1 + num2);
}

int main()
{
    sum(10);       // 错误：少传一个
    sum(10,20,30); // 错误：多传一个

    return 0;
}

五、带参数练习：打印多次内容

题目：定义一个函数，打印多次内容，打印次数由传入的参数决定。

这个函数只需要知道“打印几次”，所以只需要一个形参。 
#include <stdio.h>

void print_text(int count)
{
    for (int i = 1; i <= count; i++)
    {
        printf("I love C\n");
    }
}

int main()
{
    print_text(3);
    print_text(5);

    return 0;
}
形参数量由需求决定。需要几个不确定的数据，就定义几个形参。

六、带返回值的函数

为什么需要返回值

如果函数只是把结果打印到控制台，调用处拿不到结果。 但有些题目中，调用处还要继续使用这个结果，例如比较大小、继续计算、做判断。 这时就必须让函数把结果返回给调用处。

return 的两个作用

作用 1

结束函数。执行到 return 后，函数结束，后面的代码不会再执行。

作用 2

把 return 后面的数据返回给调用处。哪里调用，就返回到哪里。

带返回值函数格式

返回值类型 函数名(形参列表)
{
    函数体;

    return 返回值;
}

例如：

int sum(int num1, int num2)
{
    int result = num1 + num2;
    return result;
}
返回值类型必须和 return 后面的值类型对应。返回整数写 int，返回小数写 double。

带返回值函数的调用方式

方式一：用变量接收，最常用。

int result = sum(10, 20);

方式二：直接放进 printf 中打印。

printf("%d\n", sum(10, 20));
如果后面还要继续判断或计算，推荐用变量接收返回值。

七、返回值案例：比较两个人考试总分

小慧慧基础题 93 分，附加题 10 分；小丹丹基础题 87 分，附加题 9 分。判断谁的总分更高。 

#include <stdio.h>

int sum(int base, int additional)
{
    int result = base + additional;
    return result;
}

int main()
{
    int score1 = sum(93, 10);
    int score2 = sum(87, 9);

    if (score1 > score2)
    {
        printf("小慧慧的考试成绩更高\n");
    }
    else if (score1 < score2)
    {
        printf("小丹丹的考试成绩更高\n");
    }
    else
    {
        printf("两个人的考试成绩是一样的\n");
    }

    return 0;
}
这个例子的关键：函数负责计算总分，main 拿到返回值后继续判断。

八、函数的两个好处

复用性

同一个函数可以调用多次，不用重复写相同代码。

可维护性

代码出问题时，可以集中检查对应函数，不用从头到尾乱找。

九、定义函数的“三问法”

很多初学者不会定义函数，主要卡在两个问题： 形参到底写几个？返回值类型到底写 void 还是具体类型？ 解决方法就是定义函数前问自己三个问题。

问题 决定什么
我定义函数是为了干什么？ 决定函数体
我干这件事需要什么数据？ 决定形参
做完后，调用处还要不要继续使用结果？ 决定返回值类型
如果调用处还要继续使用结果，就写具体返回值类型，并 return 结果。 如果调用处不需要结果，只是执行动作，可以写 void。

十、三问法案例：比较两个长方形面积

分析

  • 函数要做什么：求长方形面积。
  • 需要什么数据：长和宽。
  • 调用处是否继续使用结果：需要，因为要比较哪个面积更大。
  • 面积可能是小数，所以返回值类型用 double。
#include <stdio.h>

double get_area(double length, double width)
{
    double area = length * width;
    return area;
}

int main()
{
    double area1 = get_area(5.3, 2.7);
    double area2 = get_area(1.5, 0.8);

    if (area1 > area2)
    {
        printf("第一个长方形更大\n");
    }
    else if (area1 < area2)
    {
        printf("第二个长方形更大\n");
    }
    else
    {
        printf("两个长方形一样大\n");
    }

    return 0;
}
get_area 表示“获取面积”。函数名最好表达功能，不要随便起名。

十一、三问法练习：比较两个圆面积

关键点

圆面积公式是： 

面积 = 3.14 * 半径 * 半径
易错点：圆周率 3.14 不需要作为形参。因为它是固定值，不是每次调用都变化的数据。

真正需要从外部传入的是半径，所以形参只需要一个： 

double radius

完整代码：

#include <stdio.h>

double get_circle_area(double radius)
{
    double area = 3.14 * radius * radius;
    return area;
}

int main()
{
    double area1 = get_circle_area(3.5);
    double area2 = get_circle_area(2.8);

    if (area1 > area2)
    {
        printf("第一个圆更大\n");
    }
    else if (area1 < area2)
    {
        printf("第二个圆更大\n");
    }
    else
    {
        printf("两个圆一样大\n");
    }

    return 0;
}
形参不是“公式里出现什么就写什么”，而是“外部不确定、需要传进来的数据”。

十二、易错点速记

  • 易错 函数定义了不会自动执行，必须调用。
  • 易错 函数调用必须写小括号，例如 sum();。
  • 易错 自定义函数不能写在 main 函数里面。
  • 易错 形参和实参必须一一对应，数量不能多也不能少。
  • 易错 多个形参之间用英文逗号隔开。
  • 易错 void 表示没有返回值；要返回结果就不能写 void。
  • 易错 return 后面的值类型要和函数返回值类型对应。
  • 易错 return 会结束函数，后面的代码不会再执行。
  • 易错 固定值通常不用当形参，比如圆周率 3.14。

十三、最重要的代码模板

1. 无参数无返回值

void 函数名()
{
    函数体;
}

2. 有参数无返回值

void 函数名(类型 参数1, 类型 参数2)
{
    函数体;
}

3. 有参数有返回值

返回值类型 函数名(类型 参数1, 类型 参数2)
{
    函数体;

    return 返回值;
}

4. 调用函数

函数名();

函数名(实参1, 实参2);

变量 = 函数名(实参1, 实参2);

十五、最终速记

函数就是把代码打包成一个功能，以后通过函数名重复使用。
形参是定义函数时用来接收数据的变量；实参是调用函数时真正传进去的数据。
如果调用处还要继续使用函数计算出来的结果，就必须使用返回值。
定义函数前先问三句话：我要做什么？需要什么数据？结果还要不要继续用？
