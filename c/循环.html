1. while 循环

while 循环的作用：当条件成立时，重复执行一段代码；条件不成立时，循环结束。

基本格式

初始化语句;

while (条件判断语句)
{
    循环体语句;
    条件控制语句;
}

其中，初始化语句写在 while 外面；条件判断语句写在小括号里；循环体和条件控制语句写在大括号里。

执行流程

先执行初始化语句，然后判断条件。如果条件成立，就执行循环体和条件控制语句。执行完一次后，再回到条件判断处继续判断。直到条件不成立，循环结束。

重点：while 小括号里写的是“继续循环的条件”，不是“结束条件”。

示例：打印 3 次内容

#include <stdio.h>

int main()
{
    int i = 1;

    while (i <= 3)
    {
        printf("IV GG\n");
        i++;
    }

    return 0;
}
注意：如果忘记 i++，i 永远是 1，条件一直成立，会造成死循环。
本节重点：while = 初始化在外，条件在小括号，循环体和条件控制在大括号；条件成立就继续，条件不成立才结束。

2. for 和 while 的区别

共同点：for 和 while 都是循环结构，运行规则本质一样：初始化 → 判断条件 → 执行循环体 → 条件控制 → 再判断。

语法层面的小区别：变量作用范围

如果变量定义在 for 小括号里，循环结束后不能继续使用。

for (int i = 1; i <= 3; i++)
{
    printf("%d\n", i);
}

// 这里不能继续使用 i

而 while 通常把变量定义在循环外面，所以循环结束后变量还可以继续用。

int i = 1;

while (i <= 3)
{
    printf("%d\n", i);
    i++;
}

printf("%d\n", i);
不要死记：变量能不能继续用，关键不是 for 或 while，而是变量定义在哪里。for 也可以把变量定义在外面。

真正重要的区别：使用习惯

情况 推荐循环 原因
知道循环次数或范围 for 开始、结束、变化规律集中在一行，清晰
不知道循环次数，只知道结束条件 while while 小括号只关注条件，更适合不确定次数

典型判断

  • 获取 1 到 100 之间的数字：范围明确，用 for。
  • 密码最多重试 3 次：次数明确，用 for。
  • 读取文件直到末尾：不知道数据数量，只知道读到末尾停止，用 while。
  • 吃包子吃到撑：不知道吃几个，只知道吃撑停止，用 while。
本节重点：知道次数或范围，用 for；不知道次数，只知道结束条件，用 while。

3. 循环算法题：判断一个数是不是 2 的幂

题目：给一个整数 n，判断它是否是 2 的幂。例如 1、2、4、8、16、32 都是 2 的幂。

核心规律

如果一个数是 2 的幂，那么它不断除以 2，最终一定会得到 1。

例如：

128 / 2 = 64
64 / 2 = 32
32 / 2 = 16
16 / 2 = 8
8 / 2 = 4
4 / 2 = 2
2 / 2 = 1

如果中途不能被 2 整除，就不是 2 的幂。例如：

100 / 2 = 50
50 / 2 = 25

25 不能继续被 2 整除，所以 100 不是 2 的幂。

为什么用 while

因为不知道要除多少次，只知道结束条件：要么除到 1，要么不能继续被 2 整除。

代码

#include <stdio.h>

int main()
{
    int n;

    printf("请输入一个整数：");
    scanf("%d", &n);

    if (n <= 0)
    {
        printf("no\n");
        return 0;
    }

    while (n > 1 && n % 2 == 0)
    {
        n = n / 2;
    }

    if (n == 1)
    {
        printf("yes\n");
    }
    else
    {
        printf("no\n");
    }

    return 0;
}
核心条件：while (n > 1 && n % 2 == 0)，表示 n 还没到 1，并且还能被 2 整除，就继续除。
本题重点：2 的幂不断除以 2 最终会变成 1；中途不能被 2 整除就不是。

4. 循环算法题：折纸问题

题目：珠穆朗玛峰高度约 8844430 毫米，纸张厚度 0.1 毫米。问折叠多少次后，纸张厚度超过珠峰高度。

核心规律

纸每折叠一次，厚度变成原来的 2 倍。

paper = paper * 2;

题目问的是折叠次数，所以需要一个计数器：

int count = 0;
count++;

为什么用 while

因为不知道要折多少次，只知道停止条件：纸张厚度超过珠峰高度。

结束条件：paper > height。
继续条件：paper <= height。

代码

#include <stdio.h>

int main()
{
    double height = 8844430;
    double paper = 0.1;
    int count = 0;

    while (paper <= height)
    {
        paper = paper * 2;
        count++;
    }

    printf("需要折叠 %d 次\n", count);
    printf("最终纸张厚度：%lf 毫米\n", paper);

    return 0;
}
注意：纸张厚度是 0.1，是小数，所以用 double。珠峰高度和纸张厚度单位要统一，都是毫米。
本题重点：每折一次厚度翻倍；循环一次就是折一次；最终需要折叠 27 次。

5. 循环算法题：整数反转

题目：把一个整数反转。例如 123 反转成 321。

核心思路

从右往左依次取出每一位数字，再拼接到新数字后面。

取个位：
temp = number % 10;
去掉个位：
number = number / 10;
拼接公式：rev = rev * 10 + temp;
先把原来的结果乘以 10，相当于往左挪一位，再把新数字放到个位。

代码

#include <stdio.h>

int main()
{
    int number = 123;
    int rev = 0;

    while (number != 0)
    {
        int temp = number % 10;
        number = number / 10;
        rev = rev * 10 + temp;
    }

    printf("%d\n", rev);

    return 0;
}

过程示例

number = 123, rev = 0

第1次：temp = 3, number = 12, rev = 3
第2次：temp = 2, number = 1,  rev = 32
第3次：temp = 1, number = 0,  rev = 321
本题重点：每次取个位、去个位、拼接到 rev；当 number 变成 0 时结束。

6. 循环算法题：平方根整数部分

题目：给一个非负整数 x，返回它的算术平方根，只保留整数部分，小数部分不要。

核心思路

从 1 开始一个一个试，直到找到第一个平方大于原数的数字。

例如求 17 的平方根整数部分：

1 * 1 = 1
2 * 2 = 4
3 * 3 = 9
4 * 4 = 16
5 * 5 = 25

5 的平方已经大于 17，所以答案是 5 前面的数字 4。

规律：找到第一个满足 i * i > number 的 i，答案就是 i - 1。

代码

#include <stdio.h>

int main()
{
    int number;
    int i = 1;

    printf("请输入一个非负整数：");
    scanf("%d", &number);

    while (i * i <= number)
    {
        i++;
    }

    printf("平方根的整数部分是：%d\n", i - 1);

    return 0;
}
注意：即使 number 是 16，也不是在 4 * 4 == 16 时停，而是继续到 5 * 5 > 16，最后输出 5 - 1。
本题重点：循环条件是 i * i <= number；循环结束后，答案是 i - 1。

7. 循环算法题：回文数

题目：判断一个整数是否是回文数。回文数就是正着读和倒着读都一样的数。

例如：

  • 121 正着读和倒着读都是 121，所以是回文数。
  • 123 正着读是 123，倒着读是 321，所以不是回文数。

核心思路

这题和整数反转几乎一样，只是多一步比较：

先反转数字
再和原数字比较
相等就是回文数
不相等就不是
注意：反转过程中 number 会被改成 0，所以必须提前保存原数字。

代码

#include <stdio.h>

int main()
{
    int number = 121;
    int old = number;
    int rev = 0;

    while (number != 0)
    {
        int temp = number % 10;
        number = number / 10;
        rev = rev * 10 + temp;
    }

    if (old == rev)
    {
        printf("yes\n");
    }
    else
    {
        printf("no\n");
    }

    return 0;
}
本题重点：回文数 = 反转之后仍然等于原数字；一定要用 old 保存原数字。

8. 循环算法题：不用乘除取余实现两数相除

题目：给两个整数，被除数和除数，不能使用乘法、除法、取余，只能用加法和减法，求商和余数。

计算思想

乘法可以理解成重复加法。例如：

2 * 3 = 2 + 2 + 2

除法可以理解成重复减法。例如：

11 / 3

11 - 3 = 8
8 - 3 = 5
5 - 3 = 2

现在 2 不够再减 3，所以停止。减了 3 次，所以商是 3；最后剩下 2，所以余数是 2。

核心：商是减法执行的次数，余数是最后剩下的数字。

代码

#include <stdio.h>

int main()
{
    int dividend = 11;
    int divisor = 3;
    int quotient = 0;

    if (divisor == 0)
    {
        printf("除数不能为0\n");
        return 0;
    }

    while (dividend >= divisor)
    {
        dividend = dividend - divisor;
        quotient++;
    }

    printf("商：%d\n", quotient);
    printf("余数：%d\n", dividend);

    return 0;
}
当前版本适合正整数除法。如果以后处理负数，还要额外处理符号问题。当前课程重点是理解“除法转减法”。
本题重点：不断用被除数减除数；能减几次，商就是几；最后剩下的就是余数。

9. do...while 循环

do...while：C 语言中的一种循环结构。实际开发中用得较少，但考试中会出现，尤其是选择题。

基本格式

初始化语句;

do
{
    循环体语句;
    条件控制语句;
} while (条件判断语句);
最大特点：do...while 是先执行，再判断，所以循环体至少执行一次。

打印 1 到 5

#include <stdio.h>

int main()
{
    int i = 1;

    do
    {
        printf("%d\n", i);
        i++;
    } while (i <= 5);

    return 0;
}

证明至少执行一次

#include <stdio.h>

int main()
{
    int i = 10;

    do
    {
        printf("%d\n", i);
        i++;
    } while (i <= 5);

    return 0;
}

虽然 10 <= 5 一开始就不成立，但程序仍然会输出：

10

和 for / while 对比

循环 执行顺序 第一次条件不成立时
for 先判断，再执行 一次都不执行
while 先判断，再执行 一次都不执行
do...while 先执行，再判断 至少执行一次
考试重点：看到 do...while，第一反应就是“先执行一次，再判断”。另外，最后的 while (...); 后面有分号。
本节重点：do...while 实际开发少用，但考试常考；它先执行再判断，循环体至少执行一次。
